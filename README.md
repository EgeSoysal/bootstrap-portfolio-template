# 🚀 Bootstrap Responsive Portfolio Template

[![GitHub License](https://img.shields.io/github/license/EgeSoysal/bootstrap-portfolio-template?style=flat-square&color=orange)](LICENSE)
[![Bootstrap Version](https://img.shields.io/badge/Bootstrap-v5.3.3-7952B3?style=flat-square&logo=bootstrap&logoColor=white)](https://getbootstrap.com/)
[![Tech Stack](https://img.shields.io/badge/Stack-HTML5%20%7C%20CSS3%20%7C%20JS-blue?style=flat-square)](#-tech-stack)

A modern, clean, and fully responsive portfolio template built with Bootstrap 5. It is designed for developers, creators, and photographers who want a polished one-page experience with separate sections for about, skills, projects, gallery, blog, and contact.

---

## ✨ Main Features

- 🌓 Dark / light mode toggle with a smooth visual switch
- 📱 Fully responsive layout for desktop, tablet, and mobile screens
- 🧠 Clean sections for profile, skills, projects, and contact information
- 📷 Photography gallery layout with a simple archive-style presentation
- 📝 Static blog section with English-ready content for updates and notes
- ⚡ Lightweight structure using plain HTML, CSS, and JavaScript

## 🛠️ How to Use

1. Open the project folder in your browser or run a simple local server.
2. Edit the content in the HTML files to match your personal profile, projects, and social links.
3. Replace the sample images and placeholder text with your own assets.
4. Customize colors and spacing in the shared stylesheet if needed.
